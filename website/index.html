Seaton Valley Community Council App

A companion app for residents and tourists in Seaton Valley.

An alternative to the community council website, making it quicker and easier than ever to interact with the community.

Introduction

Our app aims to provide residents of Seaton Valley with the latest information about the local area, along with a simple and efficient way to report issues or contact members of the council. We created this app with the users' needs in mind, taking inspiration from resident feedback as well as other council apps. Using this app on your android smartphone of choice should always be the faster option. By combining functionality from the website with original ideas, we have produced a comprehensive window into everything Seaton Valley.

Features

Latest

This is the section of the app which will provide users with the latest news and events in the local area. It gives a preview of each event with a "read more" button. When this button is pressed, the app will open a page in the internet browser to take the user to the corresponding page on the council website to provide more information. There is also a search function which will allow the user to search for keywords, and will provide live results.

Tweets

This page shows all tweets from the Seaton Valley Community Council Twitter account (@SeatonValleyCC), and automatically updates when new tweets are posted.

Report It

Allows the user to report issues to the community council. The user's name and email address are automatically filled in based on the settings page.

Futures

In this page, users can see previous issues of the council's magazine "Futures". In the preview, the user can see the front cover of the magazine along with a brief description of the contents. Once the user selects the edition they wish to view, a pdf reader opens with a pdf of the correct edition.

Contact Us

This page allows the user to contact Seaton Valley Council using either the phone number, email, office location or an in app message form.

Services

On this page, users can view locations of any registered allotment or playground in Seaton Valley. The user simply chooses which allotment or playground, and a map is opened with a pin dropped at that location.

Settings

This page, accessible from any other part of the app, allows the user to enter their information, use customisation features, view the privacy policy and visit this website. Information they enter can be used to fill out report and contact forms. The user can enable the enhanced readability mode to help with disabilities such as dyslexia.
